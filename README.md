# Requirement
**1-** node</br>
**2-** npm

# Usage
**1-** `npm install`</br>
**2-** Create a folder inside the target folder.</br>
**3-** Put your files in the folder you created.</br>
**4-** `npm run start`</br>
**5-** Enter the requested data during the command.</br>
**6-** New files will be created in the new folder.
